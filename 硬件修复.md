# XIAO nRF52840 Sense 固件烧录指南

重新烧录 PsyGuard 固件，使 `PDM.setGain(30)` 生效，解决麦克风增益不足导致语音识别失败的问题。

---

## 第一步：安装 Arduino IDE

如果还没装，去 [arduino.cc/en/software](https://www.arduino.cc/en/software) 下载 **Arduino IDE 2.x**，安装完打开。

---

## 第二步：添加 Seeed 板包

1. Arduino IDE → **Arduino IDE** 菜单 → **Preferences**（偏好设置）
2. 找到 **Additional boards manager URLs**，点右边的图标
3. 添加这一行：
   ```
   https://files.seeedstudio.com/arduino/package_seeeduino_boards_index.json
   ```
4. 点 OK 保存

---

## 第三步：安装板包

1. 左侧工具栏点 **板子图标**（Boards Manager）
2. 搜索 `Seeed nRF52`
3. 找到 **Seeed nRF52 mbed-enabled Boards**（注意：必须是 **mbed-enabled** 版，不是普通版）
4. 点 **Install**，等待安装完成（文件较大，需几分钟）

---

## 第四步：打开固件

1. Arduino IDE → **File → Open**
2. 选择：`psy-guard/PsyGuard-Arduino/PsyGuard/PsyGuard.ino`

---

## 第五步：选择板子和端口

1. 顶部工具栏下拉 **Select Board**
2. 选择：**Seeed XIAO BLE Sense - nRF52840**（注意要带 Sense 字样的那个）
3. 用 USB 数据线连接 XIAO 到电脑
4. 端口下拉选择对应的串口（Mac 上通常是 `/dev/cu.usbmodemXXXX`）

> 如果连上 USB 后 XIAO 没出现在端口列表，双击 XIAO 上的 Reset 按钮进入 Bootloader 模式（此时橙灯呼吸闪烁），再查看端口。

---

## 第六步：上传

点顶部的 **→ Upload**（上传）按钮，等待编译+烧录完成，看到 **Done uploading** 即成功。

---

## 验证

烧录完成后：

- 拔掉 USB，插上电池（或保持 USB 供电）
- XIAO 应亮**蓝灯**（BLE 广播中）
- 打开 iOS App，应能自动搜到并连接
- 点录音后 XIAO 亮**红灯**，说话后服务器日志出现识别文字

查看服务器日志：

```bash
python3 -c "
import paramiko
ssh = paramiko.SSHClient()
ssh.set_missing_host_key_policy(paramiko.AutoAddPolicy())
ssh.connect('150.158.146.192', username='ubuntu', password='@Nchu1234')
_, o, _ = ssh.exec_command('docker logs --tail 30 psy-guard 2>&1')
print(o.read().decode())
"
```

日志里出现 `[ASR/stream] sn=1 ... → '识别文字'` 即端到端打通。
